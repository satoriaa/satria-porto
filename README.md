# Portfolio React (Vite + Tailwind)

Proyek ini adalah website portfolio berbasis **React** dengan **Vite** dan styling menggunakan **Tailwind CSS**.

## Fitur
- Landing page dengan section perkenalan
- Navigasi antar bagian (Navbar)
- Section proyek, skill, pengalaman, dan kontak
- Animasi/experience tambahan (mis. Splash Screen)
- Integrasi statistik GitHub (GithubStats)

## Stack Teknologi
- **React**
- **Vite**
- **Tailwind CSS**

## Cara Menjalankan
1. Install dependensi:
   ```bash
   npm install
   ```
2. Jalankan mode development:
   ```bash
   npm run dev
   ```
3. Buka browser melalui URL yang diberikan oleh Vite (biasanya http://localhost:5173).

## Build untuk Produksi
```bash
npm run build
```

## Preview
```bash
npm run preview
```

## Catatan
Struktur komponen berada di `src/components/` dan halaman utama berada di `src/App.jsx`.
